# App backend

## Install

```bash
# with npm
npm install dotenv

# or with Yarn
yarn add dotenv
```

## Local development

```bash
# with npm
npm run start

# or with Yarn
yarn start
```

Server is running on `http://localhost:3000/`

### Environment configuration

List of available environment variables:

- `MONGO_USER` (required)

- `MONGO_PASSWORD` (required)

Variables should be placed `.env` file
